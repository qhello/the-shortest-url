# The Shortest URL

Technical test, done by Quentin Hello, as part of his recruiting process

## Getting started

Launch in local

```
# Make sure correct Node.js version is used
nvm install 20
nvm use 20

# Setup
npm install

# Start
npm start
```

Run tests

```
npm test
```
